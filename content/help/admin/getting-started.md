## Welcome to AMPLIFY

The AMPLIFY admin dashboard lets you manage artists, tours, and routing for the Music Declares Emergency climate action program.

---

## Logging In

1. Navigate to [`/admin/login`](/admin/login)
2. Enter your email and password
3. You'll be redirected to the dashboard

---

## Dashboard Overview

The main dashboard shows key metrics about routing activity:

- **Total Routes**: Successful redirects to org websites
- **Unique Artists**: Artists with active routing
- **Countries Reached**: Geographic distribution
- **Fallback Rate**: Percentage of visits that couldn't be routed

The **Recent Fallback Events** section shows when routing failed and why, helping you identify and fix issues quickly.

---

## Navigation

Use the navigation menu to access:

- **Dashboard**: Analytics and fallback monitoring
- **Artists**: Manage artists, send invites, view status
- **Tours**: Create and configure tours
- **Organizations**: Set country defaults and org profiles
- **Help**: Documentation (you're here!)
